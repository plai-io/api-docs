---
title: "Generate Keyword Ideas"
api: "POST /gads/generate_keyword_ideas"
description: "This endpoint generates keyword ideas related to a given input keyword. It helps users discover relevant search terms along with useful insights such as search volume, cost-per-click (CPC), and monthly trends. Ideal for planning and optimizing Google Ads campaigns based on real search behavior."
---
