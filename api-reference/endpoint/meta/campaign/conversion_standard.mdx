### Non Dynamic Ads Parameters

<ParamField body="campaignType" type="string" required>
	For Conversions Goal, campaignType is `CONVERSIONS`
</ParamField>

<ParamField body="pixelId" type="string" required>
	Facebook Pixel ID for conversion tracking, you can get the pixel id from
	[here](/api-reference/endpoint/meta/get_pixels_list)
</ParamField>

<ParamField body="conversionGoal" type="string" required>
	Campaign conversion goal type, get the list of conversion goals from
	[here](/api-reference/endpoint/meta/get_custom_conversions), use
	`CONVERSIONS` in campaignType to get the list of conversion goals.
</ParamField>
