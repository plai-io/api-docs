---
title: "Get Instagram Page Media"
api: "POST /meta/get_ig_page_media"
description: "This endpoint is used to fetch media assets (both images and videos) associated with a user's Instagram Page. It provides access to the creative library that can be used in ad campaigns, allowing users to browse and select previously uploaded media for their advertisements."
---

## Request Body

<ParamField body="userId" type="string" required>
	User Id
</ParamField>
<ParamField body="instagramId" type="string" required>
	Instagram Page ID
</ParamField>
<ParamField body="pagination" type="string" default="">
	Pagination
</ParamField>
