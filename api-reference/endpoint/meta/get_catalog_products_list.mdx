---
title: "Get Catalog Products List"
api: "POST /meta/get_catalog_products_list"
description: "This endpoint is used to get the list of products in a catalog."
---

## Request Body

<ParamField body='userId' type='string' required>
  User ID, If not already created, make a user on [create user
  page](/api-reference/endpoint/auth/create_user) first. If already created,
  retrieve the user ID from [get profile
  page](/api-reference/endpoint/auth/get_profile).
</ParamField>

<ParamField body='productSetId' type='string' required>
  The unique identifier of the product set
</ParamField>
