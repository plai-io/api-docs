---
title: "Get Page Posts"
api: "POST /meta/get_page_posts"
description: "This endpoint is used to get page posts from facebook and instagram. You can use this posts to create boost post ads."
---

## Request Body

<ParamField body="userId" type="string" required>
	User ID, If not already created, make a user on [create user
	page](/api-reference/endpoint/auth/create_user) first. If already created,
	retrieve the user ID from [get profile
	page](/api-reference/endpoint/auth/get_profile).
</ParamField>

<ParamField body="platform" type="string" required>
	The social media platform for the operation. `FACEBOOK` or `INSTAGRAM`.
</ParamField>

<ParamField body="pagination" type="string">
	Pagination
</ParamField>
