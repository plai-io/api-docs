---
title: "Update Campaign Status"
api: "POST /meta/update_campaign_status"
description: "This endpoint is used to update campaign status."
---

## Request Body

<ParamField body="userId" type="string" required>
	User ID, If not already created, make a user on [create user
	page](/api-reference/endpoint/auth/create_user) first. If already created,
	retrieve the user ID from [get profile
	page](/api-reference/endpoint/auth/get_profile).
</ParamField>

<ParamField body="campaignId" type="string" required>
	Campaign ID. You can get campaign id from [get campaign insights
	page](/api-reference/endpoint/meta/get_campaign_insights). If not created,
	create a campaign on [create campaign
	page](/api-reference/endpoint/meta/create_campaign).
</ParamField>

<ParamField body="status" type="string" required>
	Status to update the campaign to.
</ParamField>
