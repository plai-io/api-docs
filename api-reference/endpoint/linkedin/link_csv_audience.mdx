---
title: "Link CSV Audience"
api: "POST /linkedin/link_csv_audience_file"
description: "This endpoint is used to link a CSV audience to a LinkedIn."
---

## Request Body

<ParamField body="userId" type="string" required>
	User ID, If not already created, make a user on [create user
	page](/api-reference/endpoint/auth/create_user) first. If already created,
	retrieve the user ID from [get profile
	page](/api-reference/endpoint/auth/get_profile).
</ParamField>

<ParamField body="audienceId" type="string" required>
	Audience ID of the audience to link. If not already created, make a audience
	on [create audience
	page](/api-reference/endpoint/linkedin/create_csv_audience) first. If
	already created, retrieve the audience ID from [get audience
	page](/api-reference/endpoint/linkedin/get_audience_list).
</ParamField>

<ParamField body="csvFileId" type="string" required>
	Csv File ID of the CSV file to link.
</ParamField>
